# UI Boxx Skill

A interface do TaskFloww deve seguir o padrão visual Boxx.

## Direção visual

- SaaS moderno
- visual limpo
- fundo claro levemente bege
- cards brancos
- bordas grandes e arredondadas
- sombras suaves
- sidebar fixa
- header simples
- pouco uso de cores fortes
- foco em operação e produtividade

## Tailwind

Preferir:
- rounded-2xl
- rounded-3xl
- bg-white
- shadow-sm
- text-zinc-*
- border-zinc-200
- bg-[#f4f1ec]

Evitar:
- excesso de gradientes
- excesso de neon
- telas escuras como padrão
- animações desnecessárias
- componentes muito carregados
