# TaskFloww Product Rules

TaskFloww é uma plataforma de gestão operacional para agências.

## Módulos planejados

- Dashboard
- Tarefas
- Projetos
- Clientes
- Equipe
- Relatórios
- Configurações
- Kanban
- SLA
- Refações
- Produtividade

## Não é escopo

- ERP financeiro
- contas a pagar
- contas a receber
- emissão de nota fiscal
- estoque
- folha de pagamento

## Fases

Fase 2.2:
- rotas base
- navegação
- sidebar com item ativo
- páginas placeholder

Fase 3:
- Kanban visual
- colunas
- cards
- drag and drop com dnd-kit

Fase 4:
- Backend real
- PostgreSQL
- modelos
- endpoints

Fase 5:
- autenticação
- usuários
- empresas
- permissões
