# Database Rules

Banco de dados será PostgreSQL.

Não criar tabelas, migrations ou models antes da fase aprovada.

Quando chegar a fase de banco, criar primeiro:

- empresas
- usuarios
- clientes
- projetos
- tarefas
- colunas_kanban
- comentarios
- anexos
- historico_tarefas

Redis será usado futuramente para:
- filas
- cache
- notificações
- jobs assíncronos
