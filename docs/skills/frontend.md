# Skill - Frontend TaskFloww

## Padrão visual

Interface clara, compacta e moderna.

Base visual:
- Claude Design Boxx
- iClips como referência de navegação
- Linear pela limpeza visual
- ClickUp pela organização operacional

## Componentes principais

Criar componentes reutilizáveis:

- AppShell
- Sidebar
- Header
- PageHeader
- Button
- Card
- Badge
- Table
- Drawer
- Modal
- Tabs
- KanbanBoard
- KanbanColumn
- KanbanCard
- TaskDetailModal

## Kanban

Usar:
- @dnd-kit/core
- @dnd-kit/sortable
- @dnd-kit/utilities

Colunas iniciais:
- Não planejadas
- Planejadas
- Em andamento
- Aguardando aprovação
- Finalizadas

## Regras visuais

- evitar telas escuras como padrão;
- usar fundo cinza claro;
- cards brancos;
- bordas suaves;
- sombras leves;
- tipografia consistente;
- espaçamentos padronizados.
