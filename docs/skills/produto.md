# Skill - Produto TaskFloww

## Posicionamento

TaskFloww é um SaaS de gestão operacional para agências.

Não é ERP.

## Módulos principais

- Dashboard
- Gestão de Atividades
- Kanban
- Projetos
- Tarefas / Peças
- Clientes
- Agências
- Colaboradores
- Departamentos
- Workflow
- Biblioteca de Arquivos
- Relatórios Operacionais

## Relatórios importantes

- tarefas por cliente
- tarefas por colaborador
- tarefas por departamento
- tarefas atrasadas
- refações por cliente
- refações por colaborador
- tempo médio de entrega
- SLA
- carga de trabalho
- produtividade

## Fora do escopo inicial

- financeiro completo
- boletos
- notas fiscais
- contas a pagar
- contas a receber
- DRE
- contratos complexos
