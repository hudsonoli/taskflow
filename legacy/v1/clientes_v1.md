# Módulo Clientes

## Conceito

Clientes são entidades operacionais.

TaskFlow NÃO é ERP.

## O que clientes possuem

- Dados
- Endereço
- Informações Complementares
- Contatos
- Comentários
- Documentos
- Histórico

## O que clientes NÃO possuem

- Fee mensal
- Verba
- Contratos
- Integrações financeiras
- Perfil financeiro

## Endereço

Ao informar CEP:

- Logradouro
- Bairro
- Cidade
- UF
- País

devem ser preenchidos automaticamente.

Utilizar ViaCEP.

## Histórico

Registrar:

- usuário
- data
- hora
- ação
- valor anterior
- valor novo
